# SVG to JSX 🍭
> Converts SVG to valid React SVG, written in Svelte

### [Live 👈](http://svg2jsx.herokuapp.com/)

<img src="https://media.giphy.com/media/3ohzdFDUko7qK0xF4I/giphy.gif"/>

### Install ⚙️

```
yarn
```

### Start 🏃

```
yarn start:development
```

### Build 😷

```
yarn build
```
